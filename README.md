# MeoChat-APP

此项目为[MeoChat](https://github.com/Mios-dream/MoeChat)的客户端，主要提供桌面助手和助手管理

使用electron进行开发，使用原生live2d SDK，并对sdk代码添加了注释。



## 环境

- nodejs：v22.19.0

- vue: vue@3.5.21

- electron：v38.1.2

- live2d SDK：Live2D Cubism SDK Core Version 5.1.0

## 安装

```sh
npm install
```

### 开发模式启动

```sh
npm run dev:electron
```

### 构建项目

```sh
npm run build
```
